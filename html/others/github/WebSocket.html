WebSocket-Server

はじめに

おそらく読者の中には、「WebSocket Serverとはなんやねん？」となっている方もいらっしゃると思います。 本リポジトリを誰もが使いやすくするため、WebSocket Serverに関しての記事を後日書こうと思います。

リポジトリの説明

ではリポジトリの説明をします！
ほとんどの読者がこのように思っていると思います。「このリポジトリで何が出来んねん！」ズバリ言いますと、このリポジトリを用いることで、超絶簡易的なWebSocket通信をすることが出来ます。 ブラウザ上にあるボタンをクリックすると、WebSocket通信を行ってブラウザとローカルのプログラムとメッセージを送り合ったり、やり取りをすることが出来ます。
では、より具体的に見ていきます。

  1. index.jsをコンパイルします。そうするとWebSocket通信の仮想サーバが立ち上がります。node index.js
  2. 次にindex.htmlをダブルクリックで立ち上げてみてください。これでクライアント側を立ち上げたことになります。（※簡易的すぎて呆れてしまうかもしれません）
  3. ではブラウザ上で、「右クリック」→「検証」→「Console」としてみてください
  4. Console画面まで行くと、何か書いてあると思います。「WebSocket Serverの接続に成功しました」
  5. このメッセージの通り、サーバ側（node index.jsとしたJSファイル）とクライアント側（ブラウザで立ち上げているhtmlファイル）の両方の準備ができたので、WebSocket通信が行える状態になったんですね！
  6. この画面を見ながら画面左上にあるボタンをクリックしてみてください。そうするとブラウザ上のコンソール画面とnode index.jsを打ち込んだコマンドプロンプトの両方に文字が出ていることが確認できます。
以上でこのリポジトリができることは全てです。簡単でしたね。
「これで何ができるねん！」って思っているそこのあなた！このプログラムではボタンのみでしたが、この部分が入力フォームだったらどうでしょうか？
入力された文字がボタンを押された瞬間にWebSocket通信によりサーバに送られ、別のWebSocket通信を行っているブラウザにメッセージを送ることが出来ます。このようにすればチャットアプリが出来ちゃいますね！
